# Knowledge-OS
KnowledgeOS is mijn persoonlijke Cognitive Operating System: één centraal systeem waarin kennis, context, ideeën, beslissingen en AI samenkomen om contextverlies en het bekende "waar stond dit ook alweer?" voorgoed te verminderen. Het doel is cognitieve belasting te verlagen en toekomstige tools als modules binnen één hoofdproject te ontwikkelen.
